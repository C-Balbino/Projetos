## Análise de investimentos com Python
Este repositório contém os meus projetos de análise de investimentos utilizando Python. Os projetos incluem scripts de ETL, análise de dados e geração de relatórios.


### Projeto 1: Carteira aleatória de investimentos
O projeto gera duas carteiras de ativos financeiros, uma com o maior retorno e outra com o maior Índice de Sharpe. As carteiras são geradas aleatoriamente, a partir da composição de um índice da B3, a ser escolhido pelo usuário. Para a extração dos dados, foi utilizada a biblioteca playwright e, para a apresentação, a biblioteca streamlit. 

_(Este repositório é um projeto em expansão.)_
